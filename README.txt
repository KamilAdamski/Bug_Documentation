PL
Celem repozytorium jest zaprezentowanie znalezionych przeze mnie błędów oraz przedstawienie ich dokumentacji.
Dokumentacja każdego pojedynczego błędu jest dostępna w wersji polskiej oraz angielskiej(PL oraz ENG). Dodatkowo przed każdym tutułem dostępna jest liczba w [], która jest niczym innym niż numerem ID znalezionego błędu. Zgodnie z tym co wcześniej wspomniałem, każda dokumentacja występuje dwukrotnie, ale w różnych językach.

ENG
The purpose of the repository is to showcase the errors I have found and present their documentation.
The documentation for each individual error is available in both Polish and English (PL and ENG) versions. Additionally, preceding each title, there is a number in brackets [], which is nothing but the ID number of the discovered error. As mentioned earlier, each documentation appears twice, but in different languages.
